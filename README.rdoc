== Instacam

Instacam is your photo sharing social media platform. Snap pictures then upload them for all to see! If you see a picture you like, like it so that person knows!

== Tools

* Ruby on Rails
* Html/Css
* RSpec/Rails
* Capybara
* Launchy(debugging)
* Omniauth(facebook)
